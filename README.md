# Pokédex Web

Uma aplicação web interativa que consome a PokéAPI para listar e exibir os detalhes dos 151 Pokémons clássicos da primeira geração.

## 🚀 Como rodar localmente

Este projeto utiliza apenas tecnologias front-end nativas (Vanilla JS), sem necessidade de instalação de dependências complexas (Node.js, NPM, etc).

1. **Pré-requisitos:** Um navegador web atualizado.
2. **Clone o repositório:**
   ```bash
   git clone https://github.com/pvgamileira/ProjetoPokedex/
3. **Link do projeto:** https://pvgamileira.github.io/ProjetoPokedex/
